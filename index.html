About this Project

Details about the Google Prompt Wars submission

Google Prompt Wars

The Problem Statement

"Create an assistant that helps users understand the election process, timelines, and steps in an interactive and easy-to-follow way."

The Solution

This application transforms static information into an engaging, multi-faceted "Election Explorer Assistant". Here is how the challenge was solved:

  • Interactive Step-by-Step Journey: Simplifies complex processes like registering and answering FAQs, tracking the process visually.
  • Election Chatbot Assistant: A simulated AI Chatbot (lower-right FAB) provides on-demand answers to common queries (e.g., "Where is my polling booth?", "What is an EVM?").
  • Dynamic Region-Based Timelines: Allows switching between General flows and specific State schedules (e.g., Maharashtra, Delhi) to provide real-world insights.
  • Interactive Voter ID UI: Let users experience the structure of an EPIC card digitally.
  • Knowledge Base & Quiz Assessment: Ensures users understand the terms, rewarding them with visual progression paths.
  • Personalized Onboarding: Asks users context questions to dynamically update the assistant flow based on their status.
  • Uncompromised Multilingual Support: Real-time in-app localization across English, Hindi, Marathi, and Tamil to support deep regional accessibility.
  • Complete Data Isolation: No mock 'my data' usage; purely educational and modular so real government endpoints could be hooked up in the future.
